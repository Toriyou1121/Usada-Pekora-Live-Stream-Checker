# Usada-Pekora-Youtube-Live-Checker
A small native Windows desktop tool themed after Vtuber Usada Pekora. You can check her Live schedule and upcoming Lives with one click. 
The whole program was written in C with Claude Code based on Opus 4.8. A simple project for model testing in my C learning.
I will continue to update the development of it and add more interesting features. If you like it, please provide me with your valuable feedback, thanks!
